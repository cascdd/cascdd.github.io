2.在Linux下，利用 socket 技术编写程序，包含客户端和服务端，实现两者之间的信息传递。#

加分项：使用容器技术（如Docker或Podman），将socket客户端和服务端分别运行在不同的容器中，实现socket客户端和服务端之间的信息传递。

安装Docker#

打开Linux的终端，卸载旧版本#

sudo yum remove docker \
        docker-client \
              docker-client-latest \
              docker-common \
              docker-latest \
              docker-latest-logrotate \
              docker-logrotate \
              docker-engine
sudo yum install -y yum-utils  				//安装docker依赖包
sudo yum-config-manager \       			//设置docker下载地址
	--add-repo \
	https://download.docker.com/linux/centos/docker-ce.repo
sudo yum install docker-ce docker-ce-cli containerd.io  //安装docker
yum list docker-ce --showduplicates | sort -r   	//查看安装docker列表

启动docker#

sudo systemctl start docker
docker -v   						//此时可以使用docker的基本命令了，查看docker的版本
docker images  						//查看本地主机上已有镜像的基本信息

配置docker阿里云镜像加速#

sudo mkdir -p /etc/docker 				//创建目录
sudo tee /etc/docker/daemon.json <<-'EOF'		//配置镜像加速器地址
{
"registry-mirrors": ["https://rrnv06ig.mirror.aliyuncs.com"]
}
EOF

启动线程服务#

sudo systemctl daemon-reload  				//启动docker后台线程
sudo systemctl daemon-reload  				//重启docker服务

拉取python镜像做运行环境#

docker search python
docker pull python
docker images						//查看本地主机上已有镜像的基本信息
docker run -it --name server -p 12345:12345 a5d7930b60cc /bin/bash
docker run -it --name cilent -p 11223:11223 a5d7930b60cc /bin/bash

3.Hugo是一个静态网站生成器，官网为：https://gohugo.io/，查阅相关资料，在Linux下，完成以下内容：#

-配置好Hugo相关环境 -使用Markdown语法编写网站内容，至少要包含的内容：Linux课程中的学习笔记、大作业完成步骤记录，且要求有相关图片、代码或命令 -使用Hugo，将上述内容生成静态网站，部署到Web服务器（如Nginx或Apache），访问Web服务器，测试可以正常访问。

以下加分项2选1：#

-加分项1：使用容器技术（如Docker或Podman）部署上述静态网站，测试可以正常访问。 -加分项2：部署上述静态网站，使其可以公网访问，可以是自己的云服务器，或者一些免费的托管平台，如gitee pages、github pages。(我选择的)

本地搭建hugo#

mkdir hugo
tar -zxf hugo_0.107.0_Linux-64bit.tar.gz -C hugo	//解压
sudo cp ./hugo /usr/local/bin			        //复制一份hugo程序到/usr/local/bin/，不然hugo执行找不到路径
hugo version						//查看版本
hugo new site bcxc-blog					//新建站点
git clone 主题						//下载主题
vim config.toml						//修改config.toml配置文件
hugo server						//启动博客
hugo new post/first_blog.md				//编写文章

hugo本地访问#

systemctl start httpd 
systemctl status httpd
firewall-cmd --add-service=http --permanent
firewall-cmd --add-port=1313/tcp --permanent
firewall-cmd --reload
firewall-cmd --list-ports

hugo部署到github上#

hugo			生成静态网站
    hugo --theme=PaperMod --baseUrl="https://github.com/cascdd/cascdd.github.io.git" –buildDrafts
//把博客部署到 github上去(部署到远端服务器)
cd public
git init					//初始化仓库 会出现一个 .git的隐藏文件
git add .		
git commit -m "First_post"
git remote add origin https://github.com/cascdd/cascdd.github.io.git//将本地的 public 和 github仓库里的关联在public下执行
git push -u origin master
