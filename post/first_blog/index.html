Linux课堂笔记

大三上学期学习Linux的课堂笔记。
December 14, 2022 · 4 min · 712 words · 陈爱莎 | Suggest Changes

一、Linux文件和目录操作命令#

ls命令#

ls /home/kingbase 		 //输出指定目录中的文件列表
ls -a /home/kingbase		 //输出指定目录中的所有文件（含隐藏文件）
ls -l /home/kingbase 		 //输出指定目录中文件属性详情信息

pwd命令#

pwd 					//显示当前工作目录的路径

cd命令#

cd /home				//进入/home 目录
cd ..					//返回上一级目录
cd ~					//进入个人的主目录
cd -					//返回上次所在的目录
cd /home/kingbase		//使用绝对路径
cd ../bin				//使用相对路径

cp命令#

cp anaconda-ks.cfg kickstart.cfg	//在当前工作目录中，将某个文件复制一份，并定义新文件名称
cp -p anaconda-ks.cfg kickstart.cfg	//保留源文件或目录的所有属性
cp -r Documents Doc			//递归复制文件和目录

find命令#

find /data -name 'alert*.log'		//将/data 及其子目录下所有文件匹配 alter*.log 的文件列出来
find . -type f					//将当前目录及其子目录中的所有文件列出
find . -ctime 20				//将当前目录及其子目录下所有最近 20 天内更新过的文件列出
find /var/log -type f -mtime +7 -ok rm {} \;	
//查找 /var/log 目录中更改时间在 7 日以前的普通文件，并在删除之前询问它们。
find / -type f -size 0 -exec ls -l {} \;	//查找系统中所有文件长度为 0 的普通文件，并列出它们的完整路径

mkdir命令#

mkdir kingbase				//在当前工作目录中，建立一个目录文件
mkdir /tmp/kingbase			//在某文件夹下创建子文件夹
mkdir -p /dir1/dir2/dir3/dir4/dir5	//递归创建目录（即使上线目录不存在，会按目录层级自动创建目录）

mv命令#

mv file01 file02			//文件或目录改名
mv file02 /tmp/			//移动文件

rm命令#

rm /tmp/a01			//删除空文件夹
rm -rf /tmp/kingbase		//递归删除文件夹

df命令#

df -h					//带有容量单位的显示系统全部磁盘使用量情况
df -PhT				//带有文件系统类型的显示系统全部磁盘使用量情况
//-T 输出时显示文件系统类型
//-h 以容易阅读的方式显示
//-P 使用 POSIX 风格显示

touch命令#

touch file01.txt			//创建出一个指定名称的空文件

二、Linux查看文件及内容处理命令#

Linux系统文件编辑简介#

在配置 KES 数据库时，会经常对各种参数文件进行编辑，因此，学员需具备较为熟练的使用文件 内容编辑工具编辑文件的能力。

cat命令#

cat anaconda-ks.cfg		//查看某个文件的内容

more命令#

more /var/log/messages		//查看某个文件的内容
....(省略)....				//分页显示文件导航命令
: <== 这里可以等待你输入命令！
less 运行时可以输入的命令有：
空白键 ：向下翻动一页；
[pagedown]：向下翻动一页；
[pageup] ：向上翻动一页；
/字串 ：向下搜寻『字串』的功能；
?字串 ：向上搜寻『字串』的功能；
n ：重复前一个搜寻 (与 / 或 ? 有关！)
N ：反向的重复前一个搜寻 (与 / 或 ? 有关！)
q ：离开 more 这个程序；

less命令#

分页显示文件内容，more 命令的相反用法。 less /var/log/messages //分页查看指定的文件内容 ….(省略)…. //分页显示文件导航命令 : <== 这里可以等待你输入命令！ less 运行时可以输入的命令有： 空白键 ：向下翻动一页； [pagedown]：向下翻动一页； [pageup] ：向上翻动一页； /字串 ：向下搜寻『字串』的功能； ?字串 ：向上搜寻『字串』的功能； n ：重复前一个搜寻 (与 / 或 ? 有关！) N ：反向的重复前一个搜寻 (与 / 或 ? 有关！) q ：离开 less 这个程序

tail命令#

tail -n 10 -f /var/log/message		//要跟踪名为 /var/log/message 的文件的增长情况

grep命令#

grep kingbase /etc/passwd		//搜索某个文件中，包含某个关键词的内容
ps -ef|grep kingbase			//搜索某个命令输出结果中，包含某个关键词的内容
grep -ir PATH /etc/*			//忽略大小写并且递归搜索关键字

sort命令#

sort -n number.txt				//对指定的文件内容按照数字大小进行排序
sort -t : -k 3 -n /etc/passwd | more	//以冒号（：）为间隔符，对指定的文件内容按照数字大小对第 3 列进行排序
//-t 指定排序时所用的栏位分隔字符。
//-k 指定需要排序的栏位。
//-n 依照数值的大小排序。

echo命令#

echo $PATH						//输出某个变量值内容
echo kingbase|passwd --stdin kingbase	//设置 kingbase 密码

wc命令#

wc -l anaconda-ks.cfg		//统计指定文件的总行数

uniq命令#

uniq -c testfile			//统计相同内容行在文件中重复出现的次数
uniq -d testfile			//仅显示指定文件中存在一摸一样内容行的信息
uniq -u testfile			//仅显示指定文件中没有存在一摸一样内容行的信息

vi/vim命令#

vim kingbase.txt			//编辑文件使用 vi/vim 进入一般模式

三、Linux文件压缩及解压命令#

为什么要压缩、和解压文件？#

为便于管理文件，节省磁盘空间，保密等，Linux 系统通常会对文件进行压缩、打包、分发，因 此学员需具备较为熟练的使用压缩和解压文件的能力。

tar命令#

tar -czvf test.tar.gz a.c		//压缩 a.c 文件为 test.tar.gz
tar -tzvf test.tar.gz			//列出压缩文件内容
tar -xzvf test.tar.gz			//解压文件

四、系统信息显示命令#

hostname命令#

hostname				//显示当前系统的主机名
hostname -i				//显示当前系统的 IP 地址

uname命令#

uname				//显示系统内核名称
uname -a				//显示系统所有相关信息（含内核名称、主机名、版本号及硬件架构）

free命令#

free -m				//以 MB 位单位显示内存使用量信息

du命令#

du -sh /boot/initramfs-3.10.0-327.el7.x86_64.img		//查看某个文件大小
du -sh /boot							//查看目录大小

netstat命令#

netstat -anlp			//显示系统网络状态中的 UDP 连接端口号使用信息

date命令#

date					//查看时间和日期
date -s "20221101 8:30:00"	//修改时间

uptime命令#

uptime				//显示系统已经运行了多长时间

time命令#

time ls				//显示命令 ls 的时间统计结果

watch命令#

watch du -sh /data		//监测磁盘空间使用率

iostat命令#

 iostat -d 2 6			//每 2 秒报告一次，持续 6 次

top命令#

top					//以默认格式显示系统运行信息

五、用户管理命令#

useradd命令#

useradd user01			//创建并设置用户信息

usermod命令#

usermod -d /home kingbase	//修改指定用户的家目录路径

六、网络操作命令#

八、系统权限及用户授权相关命令#

chmod命令#

chmod u=rwx,g=rx,o=r test01 		//修改文件的权限，修改 test01 权限
chmod a-x test01				//将权限去掉而不改变其他已存在的权限
chmod -R o+w /install/KESV8R6	//修改目录的权限	
//-c 若该文件权限确实已经更改，才显示其更改动作。
//-f 若该文件权限无法被更改也不显示错误讯息。
//-v 显示权限变更的详细资料。
//-R 对目前目录下的所有文件与子目录进行相同的权限变更(即以递回的方式逐个变更)。

chown命令#

改变文件或目录的属主和属组。 chown kingbase file01 //修改属主

//-e 列出程序时，显示每个程序所使用的环境变量。

小结&FAQ#

常见的目录管理命令小结#

ls （英文全拼：list files）: 列出目录及文件名。
cd （英文全拼：change directory）：切换目录。
pwd （英文全拼：print work directory）：显示目前的目录。
mkdir （英文全拼：make directory）：创建一个新的目录。
rmdir （英文全拼：remove directory）：删除一个空的目录。
cp （英文全拼：copy file）: 复制文件或目录。
rm （英文全拼：remove）: 删除文件或目录。
mv （英文全拼：move file）: 移动文件与目录，或修改文件与目录的名称。

Linux系统中使用以下命令来查看#

cat 由第一行开始显示文件内容。
tac 从最后一行开始显示，可以看出 tac 是 cat 的倒着写！
nl 显示的时候，顺道输出行号！
more 一页一页的显示文件内容。
less 与 more 类似，但是比 more 更好的是，他可以往前翻页！
head 只看头几行。
tail 只看尾巴几行。
man 你可以使用 [命令]来查看各个命令的使用文档，如 ：man cp。
