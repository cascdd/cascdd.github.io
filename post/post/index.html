Linux课堂笔记

一、Linux文件和目录操作命令 ls命令 ls /home/kingbase //输出指定目录中的文件列表 ls -a /home/kingbase //输出指定目录中的所有文件（含隐藏文件） ls -l /home/kingbase //输出指定目录中文件属性详情信息 pwd命令 pwd //显示当前工作目录的路径 cd命令 cd /home //进入/home 目录 cd .. //返回上一级目录 cd ~ //进入个人的主目录 cd - //返回上次所在的目录 cd /home/kingbase //使用绝对路径 cd ../bin //使用相对路径 cp命令 cp anaconda-ks.cfg kickstart.cfg //在当前工作目录中，将某个文件复制一份，并定义新文件名称 cp -p anaconda-ks.cfg kickstart.cfg //保留源文件或目录的所有属性 cp -r Documents Doc //递归复制文件和目录 find命令 find /data -name 'alert*.log' //将/data 及其子目录下所有文件匹配 alter*.log 的文件列出来 find . -type f //将当前目录及其子目录中的所有文件列出 find . -ctime 20 //将当前目录及其子目录下所有最近 20 天内更新过的文件列出 find /var/log -type f -mtime +7 -ok rm {} \; //查找 /var/log 目录中更改时间在 7 日以前的普通文件，并在删除之前询问它们。 find / -type f -size 0 -exec ls -l {} \; //查找系统中所有文件长度为 0 的普通文件，并列出它们的完整路径 mkdir命令 mkdir kingbase //在当前工作目录中，建立一个目录文件 mkdir /tmp/kingbase //在某文件夹下创建子文件夹 mkdir -p /dir1/dir2/dir3/dir4/dir5 //递归创建目录（即使上线目录不存在，会按目录层级自动创建目录） mv命令 mv file01 file02 //文件或目录改名 mv file02 /tmp/ //移动文件 rm命令 rm /tmp/a01 //删除空文件夹 rm -rf /tmp/kingbase //递归删除文件夹 df命令 df -h //带有容量单位的显示系统全部磁盘使用量情况 df -PhT //带有文件系统类型的显示系统全部磁盘使用量情况 //-T 输出时显示文件系统类型 //-h 以容易阅读的方式显示 //-P 使用 POSIX 风格显示 touch命令 touch file01....
